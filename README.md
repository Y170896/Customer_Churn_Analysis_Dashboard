# Customer_Churn_Analysis_Dashboard
Power BI dashboard analyzing customer churn, retention and revenue insights
Customer Churn Analysis & Retention Dashboard

Project Overview
This project focuses on analyzing customer churn behavior using Power BI. The dashboard identifies key churn drivers, customer segments, and revenue contribution to support business decision-making.
   
Key Insights
- High churn observed in month-to-month contract customers
- Customers without tech support show higher churn rates
- High monthly charges correlate with increased churn
- High-value customers contribute the most revenue

Dashboard Features
- Executive Summary with KPIs
- Churn Rate vs Target (Gauge Chart)
- Customer Segmentation Analysis
- Revenue Contribution by Segment
- Churn Trend Analysis
- Impact of Tech Support on Churn

Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization Techniques
